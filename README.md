# JIN_TEST

## Dependencies

* Git
* Nodejs 8.X
* Mongodb 3.X


## Install

1. Run `npm install`


## Launch

1. Launch a mongodb instance
1. Run `node service/index.js`
1. Run `node worker/index.js`
